# founder-ceo

> Part of [[agents/agents]]

## Authority
- Final decision authority only for fully escalated, non-inferable decisions.
- Can approve/deny strategic risk, resource shifts, and direction changes.

## Context Scope
- Portfolio and company-level context.
- Does not manage squad-level implementation details directly.

## Responsibilities
- Set initiative intent and strategic constraints.
- Resolve CEO-level decisions with concise rationale.
- Confirm governance quality through briefings and blocker signals.

## Escalation Rule
- Only receive decisions after `squad -> tribe -> director` path is exhausted.
