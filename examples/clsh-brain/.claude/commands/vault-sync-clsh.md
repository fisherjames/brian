1. Read all open tasks and recent changes
2. Check each department's vault files for outdated information
3. Update status fields, TODO lists, and cross-references
4. Ensure all wikilinks are valid (no broken references)
5. Output a sync summary:
   - **Files updated**: List of vault files that were modified
   - **Status changes**: What changed from previous state
   - **New TODOs**: Any new action items discovered
   - **Broken links**: Any wikilinks pointing to non-existent files
